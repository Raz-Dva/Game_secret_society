Реализовать мини-игру по поиску собак на игровом поле.

Таймлайн:
1. показывается заставка с приглашением к игре
2. по таймауту, совершается переход в игру
3. отображается игровое поле
4. пользователь кликает/тапает по изображениям собак (происходит анимация)
5. когда найдены все собаки, отображается окно с поздравлением

Требования:
- руководствоваться вложенным видео
- использовать Phaser (Pixi.js)
- в результате сборки должен получится один HTML-файл, с встроенными в него всеми ресурсами, который можно просмотреть через браузер
- при нажатии на кнопку "Play Now" происходит переход на сайт https://www.g5e.com/
- должны поддерживаться горизонтальная и вертикальная ориентации
- будет плюсом использование сборщика (например, webpack)
- необходимо предоставить исходные коды игры в виде архива или ссылки на github или другой аналогичный ресурс

Дополнительные ресурсы (должны быть предоставлены вместе с данным документом):
/assets - папка с изображениями
/gameplay.mp4 - пример рабочей игры
